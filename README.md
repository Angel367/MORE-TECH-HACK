# MORE-TECH-HACK

## Запуск
1. Склонируйте репозиторий:
```bash
git clone https://github.com/Angel367/MORE-TECH-HACK.git
```
2. Перейдите в папку с репозиторием:
```bash
cd MORE-TECH-HACK
```
3. Запустите docker-compose:
```bash
docker-compose up
```
